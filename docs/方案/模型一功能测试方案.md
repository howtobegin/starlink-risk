# 模型一多场景测试

## 测试要点

1. 单条件匹配是否正常运作
2. 多条件匹配（组合逻辑表达式）是正常运作
3. 无事件属性过滤的规则是否正常运作
4. 有多个事件属性（and连接）过滤的规则是否正常运作
5. 预警间隔是否正常运作

## 测试场景

### 场景一

要求：测试单条件、无事件属性过滤规则至少三轮预警
